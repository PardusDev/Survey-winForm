# Survey-winForm
This project is a project that was made approximately 2.5 years ago. It is similar to Google Form, allowing the creation of customizable surveys, written in C#, and capable of using a MySQL remote server as its database.


Feel free to customize and modify the theme and components to suit your personal style and branding preferences.

## Installation
1. Clone the repository
2. Open the project in Visual Studio
3. Run the project

## Usage
1. Login with the default admin account:
   - Username: Pardus
   - Password: 1

2. Create a new survey
3. Add questions to the survey

## Features
- Login system
- Admin and user roles
- Different types of questions (choice, checkbox, text, dropdown menu, evaluation)
- Create surveys
- Add questions to surveys
- Answer surveys
- Edit answered surveys
